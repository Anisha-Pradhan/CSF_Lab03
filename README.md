Aim of the Lab
To develop a clear understanding of control structures in Python by implementing conditional statements, loops, break and continue statements, and to learn how to define and invoke functions for solving programming problems in a structured and efficient manner.

These programs are designed to build basic programming skills in Python. They cover the use of conditional statements to make decisions, loops to perform repetitive tasks, and break/continue statements to control loop execution. The programs also introduce functions, which help organize code into reusable blocks for better structure and clarity. Finally, integrated programs combine these concepts to solve problems more efficiently, such as creating a calculator or checking numbers systematically.
